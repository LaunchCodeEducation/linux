## Major Concepts & Key Terminology

- `Caddy` HTTP, HTTPS, and reverse proxy server
    - `Caddyfile`: config file required for caddy
    - `caddy reload`
- `NGINX` - HTTP and reverse proxy server
    - `.conf file`: config file required for nginx
    - `nginx -s reload`
