### What is a Crontab?

{{% expand "Click Here for Answer" %}}

{{% /expand %}}

### How do you list the current user's Crontab?

{{% expand "Click Here for Answer" %}}
```bash
crontab -l
```
{{% /expand %}}

### How do you edit the current user's Crontab?

{{% expand "Click Here for Answer" %}}
```bash
crontab -e
```
